## 项目概述 python533


## 技术栈

- **后端**：Python 3
- **Web框架**：Django
- **数据采集**：爬虫技术

## 功能模块

### 管理员功能

- **登录认证**：管理员登录模块
- **用户管理**：查看、修改、删除、搜索用户信息
  - 用户权限设置
  - 用户创建
- **房屋管理**：查看、删除房屋信息

### 用户功能

- **注册登录**：用户账号注册与登录
- **首页浏览**：查看最新房价预测信息、房屋信息
  - **房屋分析**：查看房屋信息的饼状图展示
- **个人信息**：查看与修改个人信息
  - 密码修改

## 系统角色

- **管理员**：负责用户管理和房屋信息管理
- **普通用户**：注册、登录、使用系统预测和分析房屋信息

## 运行环境

- Python 3.x
- Django框架
- 建议使用PostgreSQL数据库
- 适用于Linux或Windows服务器环境

## 部署步骤

1. 配置Python环境
2. 安装Django框架
3. 配置数据库
4. 运行系统初始化脚本
5. 启动Django应用
